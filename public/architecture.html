🏗️ How It Works
Daromadchi is a multi-marketplace seller dashboard that syncs orders, inventory, and prices across Uzum Market, Yandex Market, and Wildberries. The Web App provides sellers with real-time analytics and management tools, while the Chrome Extension automates price/stock updates. All data flows through our API Routes to a PostgreSQL database, with read-only marketplace integrations (NO writes allowed). Authentication via NextAuth.js v5 with JWT tokens and email verification (6-digit code via SMTP on signup). Self-hosted on Hetzner VPS with Nginx reverse proxy.
