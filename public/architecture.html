Daromadchi — Multi-Marketplace Seller Dashboard
Syncs orders, inventory, and prices across Uzum Market, Yandex Market, and Wildberries. Web dashboard + Chrome Extension for seller analytics and automation. All marketplace integrations are strictly read-only — no writes allowed. Auth via NextAuth.js v5 (JWT + email verification). Self-hosted on Hetzner VPS.
